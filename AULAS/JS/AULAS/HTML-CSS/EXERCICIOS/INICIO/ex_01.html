Remapeamento de ECU.

O remapeamento de ECU ou "remap" é uma maneria relativamente simples de se conseguir mais potencia de um veiculo.
Apenas alternando alguns codigos no sistema original do carro ja é possivel se obter resultados sem o uso de modificaçoes externas.
